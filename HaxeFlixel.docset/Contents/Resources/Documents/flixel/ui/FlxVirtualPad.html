package flixel.ui

class FlxVirtualPad

extends FlxTypedSpriteGroup<FlxSprite>

A gamepad which contains 4 directional buttons and 4 action buttons. It's easy to set the callbacks and to customize the layout.

@author Ka Wing Chin

Constructor

new(?DPad:FlxDPadMode, ?Action:FlxActionMode)

Create a gamepad which contains 4 directional buttons and 4 action buttons.

Parameters:

DPadMode

The D-Pad mode. FULL for example.

ActionMode

The action buttons mode. A_B_C for example.

Variables

actions:FlxSpriteGroup

Group of action buttons.

buttonA:FlxButton

buttonB:FlxButton

buttonC:FlxButton

buttonDown:FlxButton

buttonLeft:FlxButton

buttonRight:FlxButton

buttonUp:FlxButton

buttonX:FlxButton

buttonY:FlxButton

dPad:FlxSpriteGroup

Group of directions buttons.

Methods

createButton(X:Float, Y:Float, Width:Int, Height:Int, Graphic:String, ?OnClick:Void‑>Void):FlxButton

Parameters:

X

The x-position of the button.

Y

The y-position of the button.

Width

The width of the button.

Height

The height of the button.

Graphic

The image of the button. It must contains 3 frames (NORMAL, HIGHLIGHT, PRESSED).

Callback

The callback for the button.

Returns:

The button

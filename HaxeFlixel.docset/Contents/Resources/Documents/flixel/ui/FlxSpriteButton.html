package flixel.ui

class FlxSpriteButton

extends FlxTypedButton_flixel_FlxSprite

implements IFlxInput

A simple button class that calls a function when clicked by the mouse.

Constructor

new(X:Float = 0, Y:Float = 0, ?Label:FlxSprite, ?OnClick:Void‑>Void)

Creates a new FlxButton object with a gray background and a callback function on the UI thread.

Parameters:

X

The x position of the button.

Y

The y position of the button.

Text

The text that you want to appear on the button.

OnClick

The function to call whenever the button is clicked.

size

color

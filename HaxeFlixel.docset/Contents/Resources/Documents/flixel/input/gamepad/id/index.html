package flixel.input.gamepad.id

LogitechID

IDs for Logitech controllers (key codes based on Cordless Rumblepad 2)

MFiID

IDs for MFi controllers

MayflashWiiRemoteID

WiiRemote hardware input ID's when using "Mode 3" of the MayFlash DolphinBar accessory

OUYAID

IDs for OUYA controllers

PS4ID

IDs for PlayStation 4 controllers

PSVitaID

Native PSVita input values. (The only way to use these is to actually be using a PSVita with the upcoming openfl vita target!)

WiiRemoteID

WiiRemote hardware input ID's when using the device directly Hardware ID's: "Nintendo RVL-CNT-01-TR" and "Nintendo RVL-CNT-01" -- the latter does not have the Motion-Plus attachment

XInputID

IDs for generic XInput controllers

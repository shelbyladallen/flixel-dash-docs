package flixel.addons.nape

class FlxNapeSprite

extends FlxSprite

FlxNapeSprite consists of a FlxSprite with a physics body. During the simulation, the sprite follows the physics body position and rotation.

By default, a rectangular physics body is created upon construction in createRectangularBody().

@author TiagoLr ( ~~~ProG4mr~~~ )

Constructor

new(X:Float = 0, Y:Float = 0, ?SimpleGraphic:FlxGraphicAsset, CreateRectangularBody:Bool = true, EnablePhysics:Bool = true)

Creates a FlxNapeSprite with an optional physics body. At each step, the physics are updated, and so is the position and rotation of the sprite to match the bodys position and rotation values. By default, a physics body with a rectangular shape will be created for the sprite's graphic. You can override this functionality and add a premade body of your own (see addPremadeBody()).

Parameters:

X

The initial x position of the sprite.

Y

The initial y position of the sprite.

SimpleGraphic

The graphic you want to display (OPTIONAL - for simple stuff only, do NOT use for animated images!).

CreateRectangularBody

Whether to create a rectangular body for this sprite (use false if you want to add a custom body).

EnablePhysics

Whether to enable physics simulation for the body (by adding it to the space)

Variables

body:Body

The physics body associated with this sprite.

physicsEnabled:Bool = false

Enables/disables this sprite's physics body in simulations by adding/removing it from the space.

Methods

addPremadeBody(NewBody:Body):Void

Makes it easier to add a physics body of your own to this sprite by setting its position, space and material for you.

Parameters:

NewBody

The new physics body replacing the old one.

createCircularBody(Radius:Float = 16, ?_Type:BodyType):Void

Creates a circular physics body for this sprite.

Parameters:

Radius

The radius of the circle-shaped body - 16 by default

_Type

The BodyType of the physics body. Optional, DYNAMIC by default.

createRectangularBody(Width:Float = 0, Height:Float = 0, ?_Type:BodyType):Void

Default method to create the physics body used by this sprite in shape of a rectangle. Override this method to create your own physics body! Call this method after calling makeGraphics() or loadGraphic() to update the body size.

Parameters:

Width

The width of the rectangle. Uses frameWidth if <= 0.

Height

The height of the rectangle. Uses frameHeight if <= 0.

_Type

The BodyType of the physics body. Optional, DYNAMIC by default.

destroy():Void

WARNING: This will remove this sprite entirely. Use kill() if you want to disable it temporarily only and reset() it later to revive it. Override this function to null out variables or manually call destroy() on class members if necessary. Don't forget to call super.destroy()!

destroyPhysObjects():Void

Destroys the physics main body.

drawDebug():Void

Hide debug outline on physics sprites if the physics debug shapes are turned on

kill():Void

Handy function for "killing" game objects. Default behavior is to flag them as nonexistent AND dead.

revive():Void

Handy function for bringing game objects "back to life". Just sets alive and exists back to true. In practice, this function is most often called by FlxObject.reset().

setBodyMaterial(Elasticity:Float = 1, DynamicFriction:Float = 0.2, StaticFriction:Float = 0.4, Density:Float = 1, RotationFriction:Float = 0.001):Void

Shortcut method to set/change the physics body material.

Parameters:

Elasticity

Elasticity of material.

DynamicFriction

Coeffecient of dynamic friction for material.

StaticFriction

Coeffecient of static friction for material.

Density

Density of this Material.

RotationFriction

Coeffecient of rolling friction for circle interactions.

inline setDrag(LinearDrag:Float = 1, AngularDrag:Float = 1):Void

Nape requires fluid spaces to add empty space linear drag and angular drag. This provides a simple drag alternative. Set any values to linearDrag or angularDrag to activate this feature for this object.

Parameters:

LinearDrag

Typical value 0.96 (1 = no drag).

AngularDrag

Typical value 0.96 (1 = no drag);

setPosition(X:Float = 0, Y:Float = 0):Void

Helper function to set the coordinates of this object. Handy since it only requires one line of code.

Parameters:

X

The new x position

Y

The new y position

update(elapsed:Float):Void

Override core physics velocity etc

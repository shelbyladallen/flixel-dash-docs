Methods

quickSetup(title:String, body:String, button_labels:Array<String>):Void

Assuming you use the default format, puts this information in the popup. This function ONLY works if you are using the default_popup.xml, OR your custom xml contains the following assets:

	 * 2 texts, ids: "title","body"
	 * 3 buttons, ids: "btn0","btn1","btn2"
	 * 3 modes, ids: "1btn","2btn","3btn"

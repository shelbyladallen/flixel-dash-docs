package flixel.addons.ui

class FlxUIState

extends FlxTransitionableState

implements IFlxUIState

implements IEventGetter

This is a simple extension of FlxState that does two things: 1) It implements the IFlxUIState interface 2) Automatically creates a FlxUI objects from a single string id

Usage:

 * Create a class that extends FlxUIState, override create, and 
 * before you call super.create(), set _xml_id to the string id
 * of the corresponding UI xml file (leave off the extension).

package flixel.addons.effects

class FlxClothSprite

extends FlxSprite

A FlxSprite that draw it's frame in a mesh and behave like a cloth.

@author adrianulima

Constructor

new(X:Float = 0, Y:Float = 0, ?SimpleGraphic:FlxGraphicAsset, Columns:Int = 0, Rows:Int = 0, PinnedSide:Int = FlxObject.UP, CrossingConstraints:Bool = false)

Creates a FlxClothSprite at a specified position with a specified one-frame graphic. If none is provided, a 16x16 image of the HaxeFlixel logo is used.

Parameters:

X

The initial X position of the sprite.

Y

The initial Y position of the sprite.

SimpleGraphic

The graphic you want to display (OPTIONAL - for simple stuff only, do NOT use for animated images!).

Columns

Number of columns of the created mesh.

Rows

Number of rows of the created mesh.

PinnedSide

The pinned side that points are not affected by wind or velocity. Use FlxObject.UP, DOWN, LEFT, RIGHT, NONE, ANY, etc.

Variables

colors:DrawData<Int>

read onlycolumns:Int

Number of columns of the mesh. To set it you must use setMesh().

read onlyconstraints:Array<FlxClothConstraint> = []

An array containing all vertices conections.

crossingConstraints:Bool = false

Adds two extra constraints crossing the squares to make the mesh more rigid. Need to call setMesh() to update.

read onlyheightInTiles:Float

The height of mesh squares. To set it you must use setMesh().

iterations:Int = 3

How many iterations will do on constraints for each update. Bigger number make constraint more strong and mesh more rigid.

read onlymeshFriction:FlxPoint = FlxPoint.get(0.99, 0.99)

Determines how quickly the mesh points come to rest.

read onlymeshPixels:BitmapData

The actual Flash BitmapData object representing the current display state of the modified framePixels.

read onlymeshScale:FlxPoint = FlxPoint.get(1, 1)

Change the size of the mesh related to the original frame. Need to call setMesh() to update.

read onlymeshVelocity:FlxPoint = FlxPoint.get()

An extra velocity applied to mesh (in pixels per second). Use to simulate gravity or wind.

pinnedSide:Int

Bit field of flags (use with FlxObject.UP, DOWN, LEFT, RIGHT, NONE, ANY, etc) indicating pinned side. Use bitwise operators to check the values stored here.

read onlypoints:Array<FlxClothPoint> = []

An array containing all vertices of the mesh, they are indexed row by row.

read onlyrows:Int

Number of rows of the mesh. To set it you must use setMesh().

read onlywidthInTiles:Float

The width of mesh squares. To set it you must use setMesh().

Methods

destroy():Void

WARNING: This will remove this sprite entirely. Use kill() if you want to disable it temporarily only and reset() it later to revive it. Used to clean up memory.

setMesh(columns:Int = 0, rows:Int = 0, meshPixelsWidth:Int = 0, meshPixelsHeight:Int = 0, ?pinned:Array<Int>):Void

Sets mesh points and constraints.

Parameters:

columns

Number of columns of the created mesh.

rows

Number of rows of the created mesh.

meshPixelsWidth

Optional, specify the width of the bitmapData where the mesh will be drawn. It uses frameWidth by default, but when mesh state is bigger, a new BitmapData is created.

meshPixelsHeight

Optional, specify the height of the bitmapData where the mesh will be drawn. It uses frameHeight by default, but when mesh state is bigger, a new BitmapData is created.

pinned

Indices of pinned points that are not affected by wind or velocity.

update(elapsed:Float):Void

Core update loop

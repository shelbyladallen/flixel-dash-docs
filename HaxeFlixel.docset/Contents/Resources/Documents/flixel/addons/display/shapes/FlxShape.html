package flixel.addons.display.shapes

class FlxShape

extends FlxSprite

extended by FlxShapeArrow, FlxShapeBox, FlxShapeCircle, FlxShapeCross, FlxShapeDonut, FlxShapeLine, FlxShapeSquareDonut

A convenience class for wrapping vector shape drawing in FlxSprites, all ready to go. Don't use this class by itself -- use things like FlxLine, FlxCircle, that extend it @author Lars A. Doucet

Constructor

new(X:Float, Y:Float, CanvasWidth:Float, CanvasHeight:Float, LineStyle_:LineStyle, FillColor:FlxColor, TrueWidth:Float = 0, TrueHeight:Float = 0)

(You should never instantiate this directly, only call it as a super) Creates a Shape wrapped in a FlxSprite

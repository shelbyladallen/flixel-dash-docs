package flixel.addons.display

class FlxBackdrop

extends FlxSprite

Used for showing infinitely scrolling backgrounds. @author Chevy Ray

Constructor

new(?Graphic:FlxGraphicAsset, ScrollX:Float = 1, ScrollY:Float = 1, RepeatX:Bool = true, RepeatY:Bool = true, SpaceX:Int = 0, SpaceY:Int = 0)

Creates an instance of the FlxBackdrop class, used to create infinitely scrolling backgrounds.

Parameters:

Graphic

The image you want to use for the backdrop.

ScrollX

Scrollrate on the X axis.

ScrollY

Scrollrate on the Y axis.

RepeatX

If the backdrop should repeat on the X axis.

RepeatY

If the backdrop should repeat on the Y axis.

SpaceX

Amount of spacing between tiles on the X axis

SpaceY

Amount of spacing between tiles on the Y axis

Variables

useScaleHack:Bool = true

Try to eliminate 1 px gap between tiles in tile render mode by increasing tile scale, so the tile will look one pixel wider than it is.

Methods

loadFrame(Frame:FlxFrame):FlxBackdrop
